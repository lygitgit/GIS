    <li><strong>信息栏 (Info Panel)：</strong> 展示详细的地理信息，如坐标、区域名称和相关信息。</li>
    <li><strong>事件显示栏 (Event Display)：</strong> 记录用户交互和系统事件，如标注点击等操作。</li>
